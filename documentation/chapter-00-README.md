# Finastriva Engineering Documentation

Dokumentasi ini menjelaskan proses pembangunan sistem **Finastriva** dari awal hingga sistem dapat menyimpan data secara **persistent menggunakan PostgreSQL**.

Dokumentasi disusun dalam bentuk **chapter** agar alur pembelajaran lebih jelas dan mudah diikuti secara bertahap.

Setiap chapter menjelaskan satu bagian sistem secara teknis, mulai dari setup environment hingga mekanisme koneksi backend dengan database.

---

## 📚 Daftar Chapter

### Chapter 1 — Project Setup

Menjelaskan setup awal project backend dan environment yang dibutuhkan untuk menjalankan sistem.

---

## 🎯 Tujuan Dokumentasi

Dokumentasi ini dibuat untuk:

* Menjelaskan **mekanik sistem backend secara jelas**
* Memberikan **alur pembelajaran yang terstruktur**
* Menjadi referensi teknis untuk pengembangan Finastriva di masa depan

---

## 📌 Catatan

Dokumentasi ini akan terus diperbarui seiring berkembangnya fitur dan arsitektur sistem Finastriva.
